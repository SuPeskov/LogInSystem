  • 5.3.9.1. Монтаж рам парапетов.

    Необходимая документация для выполнения работ: Разделы КР, КС.

    Производимые операции:

      Чтобы не повредить ветрозащиту модуля на этапе устройства кровли, уложить на модуль временные мостки из доски тольщиной не менее 28мм. Выполняя монтажные работы перемещаться по модулю по временным мосткам.
    1. Собрать конструкции рам парапетов. Установить согласно проекта по периметру модуля, выдерживая проектные отступы. Крепить на саморезы ЖЦ 6х80 с шагом 300мм.

    Контрольные параметры:

    • Совпадение с проектыми размерами: 0 -3мм.
    • Соотвесттвие рам габаритным размерам по проекту: +-2мм.
    • Детали рамы из массива дерева обработаны антисептическим составом.

    Необходимый инструмент и оборудование:

    • Аккумуляторный ударный шуруповерт.
    • Бита PZ3.
    • Рулетка 7,5м.
    • Правило 2м.

    Расходные материалы:

    • Саморез ЖЦ 6Х80.

  • 5.3.9.2. Монтаж контрбруса.

    Необходимая документация для выполнения работ: Разделы КР, КС.

    Производимые операции:

    1. Подготовить контрбрус. 5.1.1.1. Контрбрус
    2. Выполнить монтаж контрбруса, выдерживая проектный уклон. Крепить на саморезы ЖЦ 6х120 с шагом 500мм. В контрбрусе высотой более 100мм просверлить отверстие диаметром 10мм на глубину до середины контрбруса, чтобы заглубить крепежный саморез.

    Контрольные параметры:

    • Совпадение с проектыми размерами: 0 -3мм.
    • Совпадение уклона с проектом.
    • Контрбрус прижат к мембране через уплотнительную ленту.
    • Контрбрус обработан антисептическим составом.

    Необходимый инструмент и оборудование:

    • Аккумуляторный ударный шуруповерт.
    • Бита PZ3.
    • Аккумуляторный шуруповерт с зажимным патроном.
    • Сверло по дереву d 6х200мм.
    • Сверло по дереву d 10х100мм.
    • Рулетка 5м.
    • Правило 2м.

    Расходные материалы:

    • Саморез ЖЦ 6х120.

  • 5.3.9.3. Монтаж обрешетки.

    Необходимая документация для выполнения работ: Разделы КР, КС.

    Производимые операции:

    1. Подготовить доски обрешетки. Убедииться, что доски обработаны антисептическим составом.
    2. Выполнить монтаж обрешетки. Крепить гвоздями 80мм к каждому контрбрусу.

    Контрольные параметры:

    • Совпадение с проектыми размерами: 0 -3мм.
    • Обрешетка обработана антисептическим составом.

    Необходимый инструмент и оборудование:

    • Пневматический гвоздезабивной пистолет.
    • Рулетка 5м.
    • Правило 2м.

    Расходные материалы:

    • Гвоздь барабанный с винтовой накаткой CNW 28/80 BKSCH.

  • 5.3.9.4. Монтаж листового материала.

    Необходимая документация для выполнения работ: Разделы КР, КС.

    Производимые операции:

    1. Уложить кровельное ОСБ поверх обрешетки. Выдержать компенсационный зазор 3мм между листами ОСБ. Крепить на кровельные оцинкованные гвозди 45мм.

    Контрольные параметры:

    • Совпадение с проектыми размерами: 0 -3мм.
    • Наличие между листами компенсационного зазора.
    • Отсутствие выступающих шляпок, остатков проволоки от кассет гвоздей.

    Необходимый инструмент и оборудование:

    • Пневматический гвоздезабивной пистолет.
    • Рулетка 5м.
    • Правило 2м.

    Расходные материалы:

    • Гвоздь барабанный с кольцевой накаткой CRN 31/45 BKRI cnk .

  • 5.3.9.5. Монтаж воронок.

    Необходимая документация для выполнения работ: Разделы КР, КС.

    Производимые операции:

    • Бита PZ2.
    • Рулетка 5м.

    Расходные материалы:

    • Саморез ПШО 4,2х25.

  • 5.3.9.6. Монтаж кровельной ПВХ мембраны.

    Необходимая документация для выполнения работ: Разделы КР, КС.

    Производимые операции:

    1. Поверхность, укрываемая ПВХ кровлей, необходимо очистить от мусора, острых элементов, способных повредить ПВХ мембрану. Убедиться, что нет выступающих шляпок гвоздей, остатков проволоки.
    2. На поверхность, укрываемую ПВХ-мембраной уложить подложку из геотекстиля. Закрепить скобами А-06. Скобы должны быть полностью утоплены в геотекстиле. Выступающие края скоб не допускаются.
    3. Последовательность укладки ПВХ мембраны описано в руководстве Схема устройства ПВХ-кровли

    Контрольные параметры:

    • Совпадение с проектыми размерами: 0 -3мм.
    • Вывод водосточной трубы не попадает на вентиляционные каналы по стене.

    Необходимый инструмент и оборудование:

    • Аккумуляторный шуруповерт с зажимным патроном.
    • Бита PZ2.
    • Рулетка 5м.

    Расходные материалы:

    • Саморез ПШО 4,2х25.
